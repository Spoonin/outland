# Полная производственная цепь: материал → передел → фабрикация → структура

> Тщательный разбор того, что реально стоит между сырьём и готовым сооружением. Мотив (игрок,
> 2026-07-09): «из слитка в реальности никто не производит сразу готовые строения — слиток нужно
> превратить в прокат». Верно: игра схлопывает **весь передел и фабрикацию** (steel → buildMaterials
> → структура), а это 3–4 отдельные стадии. Плюс важный поворот: на Марсе цепь **не земная** —
> прокат/ковка массо-запретны к завозу, доминирует аддитивка/литьё. Веб-исследование в конце.

## 1. Универсальная форма цепи (любой материал → структура)

Шесть стадий. Игра сейчас прыгает со стадии 2/3 сразу на 6 — стадии **3–5 (передел + фабрикация)
отсутствуют целиком**.

| # | Стадия | Что происходит | Пример (сталь) | В игре? |
|---|--------|----------------|----------------|---------|
| 1 | **Добыча** | сырьё из грунта/атмосферы/льда | руда/реголит | ❌ (брешь №1) |
| 2 | **Восстановление/рафинирование** | сырьё → чистый материал | оксид Fe → железо → сталь | частично (`steel` есть, но из импорта) |
| 3 | **Первичный передел** | материал → полуфабрикат (сток) | непрерывное литьё → **billet/bloom/slab** | ❌ |
| 4 | **Вторичный передел** | полуфабрикат → пригодный продукт | горячий прокат → **лист/балка/арматура/труба** | ❌ |
| 5 | **Фабрикация** | продукт → компоненты | резка/сварка/мехобработка/сборка узлов | ❌ |
| 6 | **Строительство/монтаж** | компоненты → сооружение | монтаж каркаса | ✅ (структура «из стали») |

Стадии 3–5 — это и есть «прокат и всё после него», о котором ты говоришь. Между «слитком» и
«зданием» на Земле лежит целый металлургический завод.

## 2. Земная цепь стали (эталон, веб-подтверждённый)

```
руда → [доменная печь / DRI] → чугун/губчатое железо
     → [BOF / электродуговая печь] → жидкая сталь
     → [непрерывное литьё] → billet (100–200 мм квадрат) / bloom / slab   ← полуфабрикат
     → [нагрев + горячий прокат] → лист, плита, балка, пруток, арматура, рельс, катанка   ← прокат
     → [холодный прокат / волочение / отделка] → лист/проволока точных размеров
     → [фабрикация: резка, гибка, сварка, мехобработка] → компоненты
     → [монтаж] → сооружение
```

Ключевые факты (веб): непрерывное литьё даёт «semi-finished billet/bloom/slab для последующего
проката»; billet — «первичный feedstock для прутка, катанки, проволоки, структурных профилей»;
конверсия billet в продукт — «нагрев, горячий прокат, контролируемое охлаждение». То есть billet
(полуфабрикат) ≠ готовый продукт — между ними прокатный стан.

**Оборудование стадий 3–4 огромно:** прокатный стан, МНЛЗ, нагревательные печи — тысячи тонн,
здания. Это критично для Марса (см. §4).

## 3. Цепи других материалов (та же логика, свои переделы)

- **Стекло:** кремнезём → расплав → (флоат-процесс) **листовое стекло** / (фильерное вытягивание)
  **стекловолокно**. Передел = формование листа/волокна из расплава.
- **Базальт:** расплав базальта → **базальтовое волокно** (фильеры) → арматура/композит/изоляция.
  Прямой марсианский материал (базальтовый реголит).
- **Полимеры:** C1-химия (CO₂→CO→ФТ) → мономеры → полимеризация → **гранулы (pellets)** →
  (экструзия/литьё под давлением/3D-печать) → **готовые изделия**. Передел = формовка из гранул.
- **Реголит-бетон:** реголит + связующее (сера/геополимер) → **печатная/литая конструкция** прямо.
  Здесь стадии 3–5 схлопнуты естественно (аддитивно печатается сразу форма) — см. §4.

## 4. МАРСИАНСКИЙ ПОВОРОТ — цепь НЕ земная (главный вывод исследования)

Off-Earth конвенциональный передел **массо-запретен**: прокатный стан/ковочный пресс/большая МНЛЗ
— тысячи тонн оборудования, завозить бессмысленно (веб: «каждый кг payload дорог → тяжёлое
конвенциональное оборудование экономически невозможно»). Поэтому реальная марсианская цепь
**пропускает прокат** и идёт по аддитивно-литейному пути:

```
реголит → [MRE / H₂-восстановление] → металл (ферросилиций / низкоуглерод. сталь) + O₂ побочно
        → [литьё в изложницу / атомизация в порошок / вытягивание в проволоку] → feedstock
        → [аддитивка: SLM (порошок) / WAAM-DED (проволока, лазер)] → near-net деталь
        → [лёгкая мехобработка / сварка] → компонент
        → [монтаж] → машина / сосуд давления / точный узел
```

Веб-факты:
- **MRE-ферросилиций** — самый эффективный ISRU-процесс: 6776 кг оборудования + 311 кВт →
  **25 т металла + 23.9 т O₂ в год**. Металл И кислород из одного процесса.
- Три марсианских стальных маршрута: (1) H₂-восстановление ильменита + карбонилирование →
  низкоуглеродистая сталь, (2) MRE → ферросилиций, (3) молтен-солт / FFC-Cambridge.
- Формовка off-Earth — **аддитивная**: SLM (спекание металлического порошка), лазерное
  коаксиальное вытягивание проволоки (WAAM/DED, «адаптируемо к микрогравитации»). Feedstock —
  порошок или проволока, которые сами надо изготовить из литого металла.
- Тонкая атмосфера Марса → почти нет конвективного охлаждения → **эффективность спекания/плавки
  неожиданно высока**.

**Объёмное строительство (стены/защита) металл вообще не использует** — три категории
марсианской 3D-печати:
1. **Низкотемпературная экструзия** — гигантские бетон-принтеры (геополимер/реголит-«чернила»).
2. **Среднетемпературный термопласт** — серный «марсианский бетон».
3. **Высокотемпературное спекание** — плавка/сплавление реголита солнцем/микроволнами/лазером.

Роботы копают реголит → мешают в печатные «чернила» → печатают защитные оболочки вокруг
надувных жилых модулей. **Металл резервируется под машины, сосуды давления, арматуру, точные узлы**
— не под объёмные стены.

**Вывод для дизайна:** дерево НЕ должно копировать земной прокатный стан. Марсианская цепь =
литьё + аддитивка (металл для машин/точности) ∥ реголит-печать (объём/защита). «Прокатный стан»
— дорогая импортная опция для случаев, где аддитивка не тянет (массовый сортовой прокат).

## 5. Что в игре схлопнуто (точный список брешей передела)

Между `steel`/`glass`/`polymers`/`metals` (материалы) и `buildMaterials` структур пропущено:
- **полуфабрикат** (billet/pellets/порошок/проволока) — нет;
- **передел в продукт** (прокат/экструзия/аддитивка → балка/лист/деталь) — нет;
- **фабрикация** (резка/сварка/мехобработка/сборка узлов) — нет;
- различие «объёмная структура (реголит-печать)» vs «машина/сосуд (металл+аддитивка)» — нет
  (всё строится из одного `steel`+`glass`).

Итог: сооружение возникает прямо из сырого материала — «здание из слитка», ровно как ты сказал.

## 6. Рекомендация по моделированию — слой `components` (передел одним звеном)

Не плодить 4 структуры на каждый материал (неиграбельно). Вставить **одно осмысленное звено
передела** — новый промежуточный ресурс `components` (передельный/фабрикованный продукт) и
структуру-фабрику. Принцип D-058 «механизируем стадию, только если добавляет РЕШЕНИЕ» — здесь
решение есть (фабрикационная мощность как узкое место).

### Механика
- Новый ресурс **`components`** = «пригодный передельный продукт + фабрикованные детали»
  (прокат/аддитив/сборка, свёрнуты).
- Новая структура **`fab_shop`** (фабрикационный цех): вход `metals`/`composite`/`polymers` +
  много энергии + немного `chips` (управление/точность) → выход `components`.
- **Структуры строятся из `components`, а не из сырого `steel`/`glass`.** Например,
  `habitat`: было `steel 6000 + polymers 1000 + glass 4000` → станет `components 8000 + composite
  3000` (объём — реголит, детали/узлы — components).
- **Объёмные структуры** (жильё/защита/склады) тяготеют к `composite` (реголит-печать, дёшево);
  **машинные/точные структуры** (реактор/фаб/электролиз) тяготеют к `components` (металл+аддитивка,
  дорого). Это переносит различие §4 в BOM.

### Что это даёт
1. **Чинит «здание из слитка»** — теперь между материалом и структурой есть передел (`fab_shop`).
2. **Реальное узкое место/решение** — фабрикационная мощность лимитирует темп стройки (как труд/
   энергия), новый вращающийся боттлнек (`path-to-100k.md` §3).
3. **Соединяется с P4 (средства производства):** тот же `fab_shop`/`machine_shop` делает и
   `components` для структур, и локальный `spares` — когерентно (одна фабрикационная база).
4. **Марсианский поворот честен:** `fab_shop` = аддитивно-литейный (энергия + чуть чипов);
   «импортный прокатный стан» — опция для масштаба, дорогая по массе.
5. **Ресурс всего +1** (`components`), играбельность цела.

### Фаза
Слой `components` встаёт в **P3–P4** роадмапа (`implementation-roadmap.md`): после того как
`metals`/`composite` локализованы (P1–P2), вставляется передел (P3 объёмная стройка из
`composite`, P4 фабрикация металлических `components` + локальный ЗИП). Ранние (бутстрап) структуры
можно оставить на импортных `components` (завоз готовых узлов, D-057), пока `fab_shop` не построен —
плавный переход, без слома старта.

### Что НЕ моделируем (флейвор в BOM/названиях, не структуры)
Отдельные billet/bloom/slab; горячий/холодный прокат раздельно; порошок vs проволока feedstock;
конкретные SLM/WAAM/DED; флоат-стекло vs фильеры. Всё свёрнуто в `fab_shop`→`components`. Правило:
одно звено-решение, а не имитация всего завода.

## 7. Открытые вопросы (к гриллингу)

- **Глубина передела:** одно звено `components` (рекомендация) или два (полуфабрикат `stock` +
  фабрикация `components`)? Второе точнее, но +2 ресурса и +2 структуры.
- **Импортный прокатный стан:** моделировать как дорогую импортную структуру (масса-стена честна)
  или просто разрешить импорт готовых `components` (D-057) без отдельного стана?
- **Различие объём/машина:** жёстко (структуры делятся на «реголитовые» и «металлические» классы)
  или мягко (BOM-микс `composite`+`components` у каждой)? Мягко проще и реалистичнее.
- **Аддитивка требует чипов:** делать `fab_shop` зависимым от `chips` (управление/ЧПУ) —
  реалистично (аддитивка цифровая), но добавляет хайтек-вход в базовую стройку. Сколько?
