# Outland — контекст проекта

> Корневой контекст-файл. Точка входа для любой новой сессии. Читать первым.
> Статус документа: **черновик** (навык `project-context` ещё проектируется).

## Питч / центральный тезис

**Outland** — игра/симулятор, доказывающий игроку через механику, насколько практически
невозможно создать устойчивую *автономную* колонию на Марсе численностью хотя бы
несколько тысяч человек. Ключевая идея: узкое место не выживание (холод, вакуум — решаемо
инженерно), а **замыкание промышленных циклов**. Минимальный эффективный масштаб многих
производств (полупроводники, точная механика, фарма, катализаторы) превышает численность
колонии → эти узлы нельзя локализовать → вечный импорт → «фантастическая дороговизна».

## Статус

- Стадия: дизайн подтверждён (D-001…D-039); **вертикальный срез реализован** (Фазы 0–6,
  `implementation.md`): движок (порт economy.py, 67 тестов) + UI на Lit (дашборд, манифест,
  дерево объекта, дебриф) + сейв. Срез играбелен наскозь и доходит до дебрифа с названным запасом хода.
- Платформа: **веб, TypeScript** (D-040; игра-аргумент должна шариться); UI на **Lit** (веб-компоненты).
  Аргумент количественный (граф зависимостей), убедителен только при честном расчёте → пошаговая
  эконом-стратегия (D-016).
- Находка калибровки: при Starship-экономике стена **индустриальная** (MES+внутр. стоимость
  чёрных узлов), не транспортная — Циолковский слабее, чем закладывал D-012; тезис усилен (D-037).
- Прототип (`prototype/`, throwaway, Python) воспроизводит три акта: автономия плато 67–78% (<100%),
  запас хода пригвождён к ~0.5 окна — референс для золотых тестов движка.
- Дальше: скаффолд (Vite+TS+Lit+Vitest), порт движка economy.py→TS, минимальный дашборд.

## Ключевые механики (дайджест; детали → mechanics.md)

1. **Граф зависимостей** — рекурсивное дерево: один насос разворачивается в сотни узлов.
2. **Тирания Циолковского + синодические окна** — масса как валюта, запуск раз в ~26 мес.
3. **Асимптота замыкания** — последние % автономии экспоненциально дороги; часть недостижима.
4. **Энтропия и износ** — замкнутые системы текут; вечный пол импорта на запчасти.
5. **Хрупкость малой популяции** — нет специалиста под каждую систему; потеря критична.

## Референсы для калибровки (дайджест; детали → references.md)

- **Биосфера-2** — провал по кислороду; пример незамкнутости.
- **БИОС-3 / MELiSSA / NASA ECLSS** — замкнутые системы жизнеобеспечения.
- **ISRU** — Sabatier, MOXIE; добыча ресурсов на месте.
- **Mars Direct (Зубрин)** — оптимистичный контртезис.
- **Минимальная жизнеспособная популяция** — литература по порогам.

## Свод ключевых решений (детали → decisions.md)

- **D-001** Артефакт = процессный навык `project-context`, не игра напрямую.
- **D-002** Контекст — живой документ между сессиями; навык поддерживает актуальность.
- **D-003** Источник правды — файл в git-репо: `documents/index.md` + под-документы.
- **D-004** Основная среда — Claude Code (доступ к диску); чат — режим-страховка.
- **D-005** Иерархия — 5 файлов, под-документы создаются лениво, глоссарий внутри `mechanics.md`.
- **D-007** Состав `index.md` — 6 секций + 2 дайджеста (механики, референсы) по строке на пункт.
- **D-008** Триггеры навыка — авто-контекст по `index.md` + явные фразы + границы сессии + создание с нуля.
- **D-009** Ядро игры — граф зависимостей на неравенстве «спрос ≥ MES» с агрегацией спроса вверх по дереву.
- **D-010** Темп прозрения — три акта (соблазн→сомнение→прозрение), игрока соблазняет автономия %, беда растёт тускло.
- **D-011** Импорт-эскейп на каждом узле + раскрытие спроса сверху вниз («импортируй — или осознай глубину»).
- **D-012** Двигатель ходов — синодические окна (~26 мес) + масса-валюта (Циолковский); зажим F vs капитал.
- **D-013** Износ — утечка циклов (η) + деградация капитала; хвост обслуживания растит F с задержкой.
- **D-014** MES скрыт от игрока, выводится эмпирически по реализованной удельной цене (статусы 🟢/🟡 обнаруживаемы).
- **D-015** Хрупкость популяции — у навыков тоже есть MES; вечный импорт экспертизы, единые точки отказа.
- **D-016** Жанр/loop — пошаговая эконом-стратегия; петля окна: обзор→план→бюджет→коммит→ход времени.
- **D-017** Победа переосмыслена — трагедия, а не триумф; концовки коллапс/отмена/штиль + дебриф-прозрение.
- **D-018** UI — хаб-дашборд + спицы; граф как drill-down «дерево объекта» (BOM), панель импорта, манифест окна.
- **D-019** Стартовый набор чёрных узлов — чипы и катализаторы как корневые отравители + метрология/фарма/спецсплавы.
- **D-020** Чёрный ≠ «нет элемента»: Марс даёт C/H/O/N; стена — MES + катализаторы. Полимеры/эпоксиды — флагман.
- **D-021** Старт с нуля; бюджет денежный — субсидия 1 трлн/синод; триллион покупает мало (шок ценника).
- **D-022** Мегапроекты / «ложные спасители» (термояд) — откладывают коллапс, не отменяют стену; сами кормят F.
- **D-023** Игрок сам наполняет завоз из прайс-каталога; цена = земная + масса×c → ловушка «локализуй балк, импортируй хайтек».
- **D-024** События — стохастика поверх детерминизма; вероятности от состояния; аварии пуска/пути, поломки, смертность от еды/фармы.
- **D-025** Две метрики — автономия (масса, соблазн) vs самодостаточность (запас хода, дебриф-онли). [грилл]
- **D-026** Структура — курируемый граф авторских узлов; «тир» = атрибут глубины. [грилл]
- **D-027** Единый денежный ledger; `F`=деньги; масса-бюджет `B` упразднён. [грилл]
- **D-028** MES — один закон, два гейта (масштаб процесса + конвейер навыков). [грилл]
- **D-029** Спрос = потребление + производный (агрегация §1.4 реальна). [грилл]
- **D-030** Население — рычаг игрока (завоз колонистов + шлейф `F`); рождения гейтнуты инфраструктурой. [грилл]
- **D-031** Отмена субсидии = земная инфляция + земные события (вне контроля колонии). [грилл]
- **D-032** Импорт-эскейп универсален — купить можно всё; «эмбарго» = временный шок-событие, не статус узла. [грилл]
- **D-033** Мегапроект — осознанная стройка игрока (накопление из излишков); эффект = оптимистичная проекция, потолок скрыт. [грилл]
- **D-034** Специалисты = статистические пулы навыков; bus factor виден (штат показан, риск — нет). [грилл]
- **D-035** Раскрытие тумана — бесплатное пассивное (коммит) + платная разведка (деньги+лаг, по узлу). [грилл]
- **D-036** Нарратив — диегетический фрейминг (статистика+новости+репорты), без персонажей и без интерпретации. [грилл]
- **D-037** Калибровка: стена индустриальная (MES+внутр. стоимость чёрных), не транспортная; Циолковский слабее D-012, тезис усилен; запас хода устойчиво ~0.5 окна. [калибровка]
- **D-038** Пусковая мощность — класс капитала (capex+содержание, переиспользование≈0), а не цена `c`; эфф. $/кг производный (~$1M, не $400); явный рычаг игрока; даёт D-012 реальные зубы. [калибровка]
- **D-039** Технология запуска — лестница игрока: одноразовые (тупик Циолковского, показ а не лекция) → Starship-R&D (удешевляет, не решает); «выиграть ракетную войну, проиграть индустриальную». [§4]
- **D-040** Стек: web/TS, движок (чистый TS, тестируем) ⟂ тонкий UI на Lit (веб-компоненты); старт — вертикальный срез. Детали → `implementation.md`. [реализация]
