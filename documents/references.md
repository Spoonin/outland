## 1. Марсианские ISRU-ресурсы (что доступно локально)

Основа для «белого» сырья и того, почему чёрные узлы чёрны не из-за нехватки атомов (`mechanics.md` §9.6).

- **Атмосфера:** ~95% CO₂, ~2,8% N₂, ~2% Ar, следы O₂/CO. Источник C, O, N.
- **Вода:** полярные шапки, подповерхностный лёд, гидратированные минералы (реголит). Источник H и O.
- **Реголит:** Fe, Si, Al, Mg, Ca, S и др. — базовые металлы и стекло/керамика в принципе.
- **Дефицит:** ископаемых углеводородов нет (биогенны); металлы платиновой группы рассеяны/редки
  → бьёт по катализаторам (§9.3). PGM (Pt/Pd/Rh) и на Земле дефицитны: добыча сосредоточена в
  ЮАР/России, спрос уже превышает предложение [PGM-supply].

### Ключевые ISRU-процессы

- **Sabatier:** CO₂ + 4H₂ → CH₄ + 2H₂O (метан-топливо). Каталитический.
- **Электролиз воды:** H₂O → H₂ + O₂ (водород для Sabatier, кислород для дыхания).
- **MOXIE-тип:** электролиз CO₂ → O₂ + CO (кислород на месте; демонстрировано на Perseverance).
- **RWGS + Фишер-Тропш:** CO₂ → CO → олефины/углеводороды (полимеры). Каталитический, большой MES.

## 2. Замкнутые системы жизнеобеспечения (калибровка КПД η, §5.1)

- **Биосфера-2** — провал по кислороду; пример незамкнутости больших систем.
- **БИОС-3 (СССР)** — частичное замыкание по воздуху/воде/частично еде.
- **MELiSSA (ESA)** — многоотсечная модель замкнутого цикла.
- **NASA ECLSS (МКС)** — реальные КПД: вода **98%** (с brine-processor; до него 93–94%),
  кислород из CO₂ **>95%** [ECLSS]. **Вывод для η:** простые петли (вода/воздух) замыкаются на
  0,95–0,98 — но 2–5% потери на цикл компаундятся → вечная подпитка. Сложные петли (еда, химия,
  фарма) не замыкаются вовсе (Биосфера-2) → η ≈ 0 для чёрных узлов. η не глобальный — он
  **по классу узла** (резолвит §5.7).

## 3. Тирания Циолковского → капитал доставки (калибровка, §4.2/§4.5, §10.3)

- Уравнение ракеты: Δv до LEO + TMI + посадка → экспонента топлива на кг полезной нагрузки.
- Синодический период Земля–Марс ~780 дней (~26 мес) — пусковые окна, ход игры (§4.1).
- **Стоимость кг на поверхность Марса (вилка):**
  - Falcon Heavy (2022): 16,8 т на Марс за ~$97 млн (reusable) → **~$5 800/кг** [FH].
  - Starship (оптимистичная цель SpaceX/Marspedia): **~$300–400/кг** на поверхность Марса;
    к LEO целятся <$100/кг [Marspedia, Starship].
- **Находка калибровки (D-037):** при Starship-экономике (~$400/кг) перевозка балка **дёшева** →
  тирания Циолковского как несущий зажим слабее, чем предполагал D-012. «Фантастическая
  дороговизна» исходит не от массы/доставки, а от **внутренней стоимости + недостижимого MES
  чёрных узлов** (чипы/фарма/катализаторы). Стена индустриальная, не транспортная.
- **Уточнение (D-038): $400/кг — амортизированная фантазия для Марса.** Марсианский груз бёрстен
  (~5 дней раз в 26 мес); пусковые мощности (башни/флот/криозаводы) под бёрст абсурдно велики для
  земного рынка → переиспользование ≈ 0 → простой 25/26 мес ложится на программу. Доставка — не
  цена `c`, а **класс капитала**: capex на пропускную способность `K` + содержание каждое окно
  (даже вхолостую). Эффективный $/кг **производный**: `топливо·W + содержание·K` ÷ масса. В
  прототипе (D-038) эфф. цена эмёрджентно ~$1M/кг (150–3000× топливного пола $400). Это даёт
  Циолковскому реальные зубы: зажим = капитал бёрст-мощности без переиспользования, не уравнение ракеты.

## 4. MES глубоких узлов — почему масштаб недостижим (§1.3, §9, D-045)

Ядро количественного аргумента: спрос колонии (тысячи человек) против минимального масштаба завода.
MES — **конечное, привязанное к реальности число** (D-045), огромное относительно достижимой колонии:
узел нелокализуем не потому, что запрещён, а потому что спрос ≪ MES при любом разумном населении.

- **Передовые микрочипы (§9.1):** современная фаб 3-нм класса — **$15–20 млрд** капекс; одна
  EUV-установка ~$200 млн, на фаб нужно **15–20** штук; гига-фаб даёт **>17 млн** пластин-эквивалентов
  12″ в год и обслуживает мировой рынок [Fab, TSMC]. Спрос колонии — порядки ниже точки безубыточности.
- **Сложная фарма / API (§9.4):** завод API экономичен при **200–500 т/год**; капекс от
  **$120 млн** (ретрофит) до **$1,8 млрд** (новые линии пептидов); сырьё-интермедиаты — 40–50%
  операционных затрат [API]. Сотни тонн одного API — это мировой спрос, не колониальный.
- **Катализаторы (§9.3):** зависят от PGM, дефицитных и на Земле; добыча сосредоточена
  географически [PGM-supply]. На Марсе PGM рассеяны → корневой отравитель (катализатор —
  вход в Sabatier/RWGS/фарму/удобрения).

### Выведенные MES-якоря (`graph.ts mesAnchor`, D-045)

Конечные значения, заданные движку. Узел локализуем при `спрос ≥ якорь` — для колонии в тысячи
человек спрос на порядки ниже, поэтому узлы стоят чёрными, но это вопрос масштаба, не запрета.

| Узел | mesAnchor | Обоснование |
|------|----------:|-------------|
| `pharma` | 6.7e5 | завод API ~200 т/год ÷ ~0.3 кг подушевого спроса |
| `electronics` | 1.0e6 | фаб обслуживает мировой рынок vs граммы на колонию |
| `catalyst` | 3.0e5 | PGM-дефицит (рассеяны на Марсе), но сырьё импортируемо (D-032) |
| `special_alloy` | 2.0e5 | линия спецметаллургии |
| `precision_metrology` | 2.0e5 | цепочка калибровки/метрологии |

## 5. Минимальная жизнеспособная популяция (калибровка §6)

- Литература по порогам: генетический минимум (сотни–тысячи) vs **порог избыточности навыков**
  (для Outland важнее второе — MES специальностей, D-015/D-028). Гига-фаб/фарма требуют тысяч
  узких специалистов на одну установку — конвейер обучения сам MES-гейтнут (D-028).

## 6. Контртезис

- **Mars Direct (Р. Зубрин)** — оптимистичный взгляд на ISRU и осуществимость; полезен как
  калибровочная верхняя граница оптимизма (баланс соблазна, §2). Starship-экономика §3 —
  его числовая опора; именно поэтому акцент тезиса смещён на MES, а не на доставку.
